== Description ==

Inserts a Simple And Clean Table of Contents into your posts, using the [toc].

And you can custom your toc style in the TOC Setting Page.

== Installation ==

1. Upload the `toc-for-wordpress` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

OR you can install toc-for-wordpress from the wordpress pulgin manager online.

Then just use the  [toc] to insert the Table of Contents.
